Merux Helm Charts

Helm repository for charts created and maintained by Merux, ready to be installed on Kubernetes or Rancher.

⚠️ Disclaimer: This project is maintained by community volunteers and designed for expert use. For quick and easy deployment, please follow the specific instructions included in the documentation of each individual chart.

Add Helm repository

To install the repository, just run:

helm repo add merux https://charts.merux.dev
helm repo update

Available Helm Charts

  • nextcloud-spreed

    To install the Nextcloud Spreed Signaling chart:

    helm install my-release merux/nextcloud-spreed
merux-helm-chart is maintained by merux-dev. Hosted securely on GitHub Pages.
